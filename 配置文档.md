## 项目概述

银发无忧小程序是一个为老年人和子女设计的亲情关爱小程序，包含以下主要功能：
- **长辈端**：查看时间、天气、公交信息、紧急联系人等
- **子女端**：绑定长辈、管理长辈信息、添加紧急联系人等

技术栈：
- 微信小程序原生开发
- 微信云开发（云数据库、云函数）
- 第三方API：天气服务（和风天气，可选）

## 开发环境准备

### 1. 注册微信小程序账号

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「立即注册」，选择「小程序」
3. 按照提示完成账号注册和认证
4. 登录后进入小程序后台，记录下您的 **AppID**

### 2. 安装微信开发者工具

1. 访问 [微信开发者工具下载页面](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 根据您的操作系统下载对应版本
3. 安装并启动微信开发者工具

## 微信开发者工具配置

### 1. 导入项目

1. 打开微信开发者工具
2. 点击「+」号或「导入项目」
3. 选择项目目录为当前文件夹
4. 填写项目名称（如「银发无忧」）
5. 填入您的小程序 AppID
6. 点击「导入」

### 2. 项目配置检查

导入后，检查 `project.config.json` 文件中的配置是否正确：
- `appid`：应与您的小程序 AppID 一致
- `miniprogramRoot`：应设置为 `miniprogram/`
- `cloudfunctionRoot`：应设置为 `cloudfunctions/`

## 云开发环境配置

### 1. 开通云开发

1. 在微信开发者工具顶部工具栏，点击「云开发」按钮
2. 如果是第一次使用，会提示开通云开发
3. 点击「开通」，按照提示完成开通流程
4. 选择基础版（免费）或付费版，根据需求选择

### 2. 创建云环境

1. 开通后，进入云开发控制台
2. 点击「设置」→「环境设置」
3. 点击「新建环境」
4. 填写环境名称（如「silveasy-env」）
5. 选择基础版
6. 点击「确定」创建
7. 创建完成后，**复制环境ID**（格式类似：`cloud1-xxx`）

### 3. 配置项目云环境

打开项目中的 `miniprogram/app.js` 文件，找到以下代码：

```javascript
this.globalData = {
  env: "",  // 这里填入您的云环境ID
  cloudInitialized: false
};
```

将刚才复制的云环境ID填入 `env` 字段，例如：

```javascript
this.globalData = {
  env: "cloud1-xxx",  // 替换为您的实际环境ID
  cloudInitialized: false
};
```

## 数据库配置

### 1. 创建数据库集合

在云开发控制台中，点击「数据库」，然后创建以下集合：

#### 必需的集合：

1. **users** - 用户信息表
   - 权限设置：「仅创建者可读写」
   
2. **bindings** - 绑定关系表
   - 权限设置：「所有用户可读，仅创建者可写」
   
3. **binding_codes** - 绑定码表
   - 权限设置：「仅创建者可读写」
   
4. **contacts** - 联系人表
   - 权限设置：「仅创建者可读写」

### 2. 设置数据库权限

为每个集合设置正确的权限规则：

1. 点击集合名称
2. 点击「权限设置」
3. 选择合适的权限规则
4. 点击「确定」保存

### 3. 数据库字段说明

#### users 集合字段：
```javascript
{
  _id: "自动生成",
  openid: "用户微信openid",
  nickname: "用户昵称",
  avatarUrl: "头像URL",
  phone: "手机号",
  role: "角色（elder/child）",
  createTime: Date,
  updateTime: Date
}
```

#### bindings 集合字段：
```javascript
{
  _id: "自动生成",
  childId: "子女用户ID",
  elderId: "长辈用户ID",
  relation: "关系描述",
  status: "状态（confirmed）",
  createTime: Date,
  updateTime: Date
}
```

#### binding_codes 集合字段：
```javascript
{
  _id: "自动生成",
  code: "6位绑定码",
  elderUserId: "长辈用户ID",
  expireTime: Date,
  used: false,
  createTime: Date,
  updateTime: Date
}
```

#### contacts 集合字段：
```javascript
{
  _id: "自动生成",
  userId: "所属用户ID",
  name: "联系人姓名",
  phone: "联系电话",
  tag: "标签",
  relation: "关系",
  isEmergency: false,
  createTime: Date,
  updateTime: Date
}
```

## 云函数配置

### 1. 上传云函数

项目中有两个云函数需要上传：
- `silveasyFunctions` - 主要业务逻辑云函数
- `quickstartFunctions` - 示例云函数

#### 上传步骤：

1. 在微信开发者工具左侧文件目录中，找到 `cloudfunctions` 文件夹
2. 右键点击 `silveasyFunctions` 文件夹
3. 选择「上传并部署：云端安装依赖」
4. 等待上传和部署完成（可能需要几分钟）
5. 同样的方式上传 `quickstartFunctions`（可选）

### 2. 验证云函数

1. 打开云开发控制台
2. 点击「云函数」
3. 确认 `silveasyFunctions` 已在列表中，状态为「正常」

### 3. 云函数说明

#### silveasyFunctions 支持的操作：
- `getOrCreateUser` - 获取或创建用户
- `generateBindCode` - 生成绑定码
- `bindElder` - 绑定长辈
- `unbindElder` - 解绑长辈
- `getWeather` - 获取天气信息
- `getElderContacts` - 获取长辈联系人
- `addElderContact` - 添加长辈联系人
- `updateElderContact` - 更新长辈联系人
- `deleteElderContact` - 删除长辈联系人

## 常见问题解决

### 1. 云开发未初始化错误

**错误信息**：`Cloud API isn't enabled, please call wx.cloud.init first`

**解决方法**：
1. 确认已开通云开发服务
2. 确认在 `app.js` 中正确配置了环境ID
3. 确认云环境已创建并处于正常状态

### 2. 云函数调用失败

**错误信息**：云函数调用超时或返回错误

**解决方法**：
1. 确认云函数已成功上传并部署
2. 检查云函数日志，查看具体错误信息
3. 确认云函数的权限设置正确

### 3. 数据库读写失败

**错误信息**：数据库操作无权限或失败

**解决方法**：
1. 检查数据库集合的权限设置
2. 确认用户已通过云函数正确创建
3. 检查数据库操作的代码逻辑

### 4. 页面空白

**问题**：点击按钮后页面空白

**解决方法**：
1. 查看控制台（Console）的错误信息
2. 确认云开发环境已正确配置
3. 检查页面代码是否有语法错误

## 第三方API配置（可选）

### 天气API配置

项目使用和风天气API获取天气信息，当前默认使用模拟数据。如需使用真实天气数据：

1. 访问 [和风天气官网](https://www.qweather.com/) 注册账号
2. 创建应用，获取 API Key
3. 打开 `cloudfunctions/silveasyFunctions/index.js`
4. 找到 `getWeather` 函数
5. 修改 `useMockData` 为 `false`
6. 将 `WEATHER_API_KEY` 替换为您的实际API Key
7. 重新上传并部署云函数
